# Daily crosswords

В какой-то момент один сайт, знаменитый своими играми и рецептами, спрятал свой ежедневный маленький кроссворд за пейволом.
Но API с данными о кроссворде остался незащищённым...
Это приложение позволяет обойти пейвол и решать ежедневный кроссворд б е с п л а т н о.

Реализован почти весь функционал мини кроссвордов:
- Открытие одной верной буквы по кнопке «Reveal letter»
- Проверка решения по кнопке «Check» (верные буквы после авточека будут подсвечиваться синим)
- После решения кроссворда можно поделиться с друзьями своим результатом

## Стек технологий

- Java 17+
- Spring Boot 3.x
- Gradle
- Thymeleaf
- HTML/CSS/JS
- капельку попробовал HTMX
- Docker compose

## Установка и запуск

1. Установить docker
2. Собрать и запустить контейнеры `docker compose up --build`
Будут запущены два контейнера:

nyt-pirate — Spring Boot приложение
postgres-db — PostgreSQL база данных

После запуска приложение будет доступно на порту 9090 `http://localhost:9090`
