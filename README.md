# Daily crosswords

В какой-то момент один сайт, знаменитый своими играми и рецептами, спрятал свой ежедневный маленький кроссворд за пейволом.
Но API с данными о кроссворде остался незащищённым...
Это приложение позволяет обойти пейвол и решать ежедневный кроссворд б е с п л а т н о.

Реализован почти весь функционал мини кроссвордов:
- Открытие одной верной буквы по кнопке «Reveal letter»
- Проверка решения по кнопке «Check» (верные буквы после авточека будут подсвечиваться синим)
- После преодоления кроссворда можно поделиться с друзьями своим временем решения

## Стек технологий

- Java 17+
- Spring Boot 3.x
- Gradle
- Thymeleaf
- HTML/CSS/JS

## Установка и запуск

Для запуска необходимо собрать проект 

`./gradlew clean build`

Запустить jar 

`java -jar build/libs/nytpirate-0.0.1-SNAPSHOT.jar`

При необходимости можно указать порт при запуске 

`java -jar build/libs/nytpirate-0.0.1-SNAPSHOT.jar --server.port=9090`
